Go

La Flexibilidad de Python con la rapidez del código de máquina

  2. Compilado, estática y fuertemente tipado.

Instalación


Paso1: Descargar su archivo comprimido desde https://github.com/KumbiaPHP/KumbiaPHP/


Paso2: Configurar Servidor Web

  1. #a2enmod rewrite
  2. Editar el archivo con direccion etc/apache2/default-server.conf
  3. AllowOverride None debemos de cambiarlo por AllowOverride ALL
  4. Se reinicia el servicio
    
						/etc/init.d/apache2 restart
