Go

La Flexibilidad de Python con la rapidez del código de máquina

  1. Programación imperativa, estructurada y libre.
  2. Compilado, estática y fuertemente tipado.
  3. No es orientado a objetos.
  4. Es un lenguaje moderno, por lo que bebe de lo mejor de muchos lenguajes
  5. Combina una sintaxis parecida a C con las características y facilidad de lenguajes dinámicos como Python.

Continuación...

  1. GO acierta con una sintaxis classra y concisa
  2. Go no está orientado a objetos por que no existe jerarquía de tipos pero implementa interfaces
  3. Recolector de basura eficiente, libre de latencia.
  4. No se utilizan los ; (punto y coma) para separar sentencias
  5. Las cláusulas de condición no llevan paréntesis.
  6. Soportado en Google por App Engine

Historia


  • Creado por Google.
  • Lanzado el 10 de noviembre del 2009
  • Diseñado por: Robert Giresemer, Rob Pike y Ken Thompson.
  • Influencias de C, Limbo, Modula, Newsqueak, Oberon, Pascal y Python.

Ventajas:


  • Sintaxis concisa
  • Diseñado para ser eficiente en tiempo de desarrollo.
  • Rápido en tiempo de ejecución.
  • Es un lenguaje concurrente, y con un buen soporte para paralelización.
  • Soporta tests y benchmarks de forma nativa.

Benchmarks: técnica utilizada para medir el rendimiento de un sistema o componente del mismo.

Instalación


Ubuntu: 



						$ sudo add-apt-repository ppa:gophers/go
					

						$ sudo apt-get update
					

						$ sudo apt-get install golang-stable

Windows:
Descargar del Sitio Oficial y descomprimir en C: ngo y agregar esa ruta a GOROOT y c: ngo nbin a PATH

  1. Los archivos fuentes de go tiene la extensión .go y se corren con la siguiente linea. 
    
						$ go run miprograma.go
  2. Para compilar el archivo para su distribuci´on se utiliza compile 
    
						$ go build miprograma.go
